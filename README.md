# DeepSeek Chatbot

A simple chatbot using the DeepSeek API.

## Setup

1. Create a virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Set your API key:
   ```bash
   export DEEPSEEK_API_KEY="your-api-key-here"
   ```

4. Run the app:
   ```bash
   python app.py
   ```
